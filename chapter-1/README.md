## Conceitos-chave
- Diferença entre processo **recursivo** e processo **iterativo**
- **Ordem de crescimento** (O(n), O(log n), O(n²))
- **Higher-order functions**: funções que recebem e retornam funções
- **Lambda expressions** como funções anônimas
- **Abstração procedural**: tratar funções como caixas-pretas
